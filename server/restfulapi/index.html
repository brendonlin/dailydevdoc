    REST¶

    "What is REST?" from https://restfulapi.net/

    Flask restful¶

    How to create a new api?¶

    1. Create a API app¶

    from flask_restful import Resource, Api

app = Flask(__name__)
api = Api(app)

    2. Create a view inherit from flask_restful.Resource¶

    class HelloWorld(Resource):
    def get(self):
        return {'hello': 'world'}

    3. Regist view to the urls¶

    api.add_resource(HelloWorld, '/')

    End

    Django Restframework¶

    How to create a new api?¶

    1. Serialize model in serializers.py¶

    from rest_framework import serializers
from newapp.models import ModelX

class ModelXSerializer(serializers.HyperlinkedModelSerializer):
    class Meta:
        ModelX = ModelX
        fields = ('name', 'email')

    2. Create API view in view.py¶

    from rest_framework import viewsets
from newapp.models import ModelX
from newapp.serializers import ModelXSerializer

class ModelXViewSet(viewsets.ModelViewSet):
    # Sorting can be set here.
    queryset = ModelX.objects.all().order_by('-date')
    serializer_class = ModelXSerializer

    3. Regist view to urls in urls.py¶

    from django.urls import include, path
from rest_framework import routers
from newapp import views

router = routers.DefaultRouter()
router.register('api-path', views.ModelXViewSet)

    End
