    Random¶

    Random number:

    # Evenly distributed float number in range [0,1] :
np.random.rand(d0, d1, ..., dn)
# Evenly distributed integer number:
np.random.randint(low, high=None, size=None)
# Binomial distribution:
np.random.binomial(n, p, size=None)
# Normal distribution:
np.random.normal(loc=0.0, scale=1.0, size=None)
# Standard normal distribution:
np.random.randn(d0, d1, ..., dn)

    Others

    1. Choose one:choice(a, size=None, replace=True, p=None)
    2. Random permutation of sequences (without modifying objects):permutation(x)
    3. Random permutation of sequences (with modifying objects):shuffle(x)
    4. Random replacement sampling:[AA[x] for x in np.random.randint(0,3,10)]
    5. Generate random numbers with a sum of 1:$\frac{number}{\Sigma{number}}$
    6. Generate random numbers between -1 and 1:rand([0,1])*2-1
    7. Random control flow condition: `np.rand() > 0.6'

    Tips¶

    • 0 is represent row and 1 is column. when axis=0 row will be calculate and disappear.Axes is a plural form of axis.
    • array can do $(n,m) \times m$ ,but matrix can do $(n,m) \times (m,k)$
