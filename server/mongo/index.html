    How to select collection¶

    Query, filter and limit¶

    db.inventory.find({ status: "A" }, { item: 1, status: 1 }).limit(10);

    filter is write before select columns

    Aggregate¶

    db.orders.aggregate([
  { $match: { status: "A" } },
  { $group: { _id: "$cust_id", total: { $sum: "$amount" } } }
]);

    like a pipeline , combined in a list

    Commonly used pipeline element:¶

    • match:
    {
  $match: {
    status: "A";
  }
}
    • group:
    { $group: { _id: "$cust_id", total: { $sum: "$amount" } } }
    • unwind:
    {
  $unwind: "$history";
}
    • lookup:
      {$lookup: {
  from: "anime_info",
  localField: "history.id",
  foreignField: "id",
  as: "anime"
  }
  }
    • project:
      {
  $project: {
              _id: 0,
              id: "$history.id",
  title_japanese: {
  $ifNull: ["$anime.title.native", "$anime.title.romaji"]
              },
              title_romaji: "$anime.title.romaji",
  start_date_fuzzy: "$anime.startDateFuzzy",
              popularity: "$anime.popularity",
  image_url_lge: "$anime.coverImage.large",
              image_url_med: "$anime.coverImage.large",
  genres: "$anime.genres",
              total_episodes: "$anime.episodes",
  isviewed: "\$history.view_status",
  description: ""
  }
}
    • skip and limit
      { $skip: 0 },
  { $limit: 10 }

    security¶

    Role is different from user.One user can have mutiple roles.Role is first, then users.

    Modify user role:

    db.grantRolesToUser("admin", [{ role: "root", db: "admin" }]);

    Migrate¶

    dump

    mongodump --uri "mongodb://{username}:{password}@{host}:{port}" --out {filepath}

    store

    mongorestore --host {host} --port {port} --username {username} --password {password} --db {dbname} {filepath}
